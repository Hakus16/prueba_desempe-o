# prueba_desempeño
The program works as follows: On startup, a menu of options will appear, allowing you to select the desired action.
There are six options in total: add a task, display assigned tasks, search for tasks, update a task, and delete a task. 
Once you select an option, you will be given the required information.
For example, if you choose option 1, adding a task, you will be asked to enter data such as the identifier, title, description, priority, and status.
Option 4 will ask for the task identifier; once found, you can update its data. 
Option 2 will display your assigned tasks with all their components.
Option 3 allows you to search for a specific task.
and option 5 lets you delete any task.
An example of how this program could be used in an educational institution where teachers have to assign and evaluate tasks.
